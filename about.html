Учебно-спортивное учреждение «Гродненская специализированное детско-юношеская школа олимпийского резерва № 4» основана в 1969 году.

Директорами школы с 1969 года являлись:

С 24.09.1969 Кулинкович Евгений Николаевич по 02.02.1971
С 23.02.1971 Крайнев Константин Сергеевич по 21.09.1976
С 04.10.1977 Пузыня Василий Иванович по 28.06.1983
С 01.09.1983 Шейндлин Александр Наумович по 21.10.1988
С 16.08.1988 Козел Валерий Иосифович по 11.07.2012
С 03.09.2012 Ародь Евгений Иванович по н.в.
